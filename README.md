# EV Health Monitoring System

A real-time EV Health Monitoring Dashboard built using Python and Streamlit.

## Features

* Real-time battery monitoring
* Voltage, current, and temperature tracking
* Battery health status analysis
* Auto-refresh dashboard
* CSV data logging
* Performance trend visualization
* Warning alerts for abnormal conditions

## Technologies Used

* Python
* Streamlit
* Pandas
* Streamlit Auto Refresh

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the dashboard

streamlit run dashboard.py
